# where bus?

Where is my bus? This web app tracks RapidKL bus routes and schedules.

## Features
- Interactive transit map
- Mobile first UX
- Live geolocation

## Tech stack

**Backend**
- Java Spring Boot with Maven
- GTFS static and realtime bindings (Protobuf)

**Frontend**
- Next.js (React + TypeScript)
- Tailwind CSS
